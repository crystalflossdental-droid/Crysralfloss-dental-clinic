# DentalSmile
A modern, responsive dental clinic website built to provide patients with a seamless experience for learning about services, booking appointments, and connecting with the dental team.

## 🚀 Quick Start
### Prerequisites
- Node.js 18+ installed on your machine
- npm or yarn package manager
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation & Local Development
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/your-username/dentalsmile.git
   cd dentalsmile
   ````

2. Install project dependencies:
   ```bash
   npm install
   # or if using yarn
   yarn install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   # or yarn dev
   ```

4. Access the application at:
   ```
   http://localhost:5173
   ```

## ✨ Features
- Responsive design optimized for mobile, tablet, and desktop devices
- Interactive appointment booking system with calendar integration
- Service showcase with detailed descriptions of dental treatments
- Team profile section introducing dentists and staff
- Patient testimonial carousel
- Contact form with Google Maps integration
- Multi-language support (English/Spanish)
- HIPAA-compliant patient data handling

## 🛠️ Tech Stack
- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Form Handling**: React Hook Form
- **Calendar**: FullCalendar
- **Maps**: Google Maps JavaScript API
- **Deployment**: Vercel
